400
Requete invalide
Le serveur n'a pas compris la requete. Le format ou les parametres envoyes sont incorrects.
Actions recommandees
Verifiez que vous utilisez https:// et non http:// sur le port HTTPS
Si vous utilisez l'API, controlez le format JSON de votre requete
Videz le cache de votre navigateur
401
Authentification requise
Vous devez vous connecter pour acceder a cette ressource. Votre session a peut-etre expire.
Actions recommandees
Reconnectez-vous depuis la page de login
Verifiez que votre token API est valide TC_AUTH_TOKEN
403
Acces interdit
Vous n'avez pas les droits necessaires. Le module de securite a bloque la requete.
Actions recommandees
Verifiez votre role utilisateur (admin requis pour cette page)
Verifiez le header Authorization: Bearer <token>
Cette action est peut-etre restreinte par la politique HITL
404
Page introuvable
La ressource demandee n'existe pas. Elle a peut-etre ete deplacee ou supprimee.
Actions recommandees
Verifiez l'URL — une faute de frappe est vite arrivee
Accedez au dashboard depuis la page d'accueil
Les endpoints API sont sous /api/
429
Trop de requetes
Le systeme anti-brute-force a temporairement limite vos requetes. C'est un mecanisme de securite.
Actions recommandees
Attendez 30 secondes avant de reessayer
Limite par defaut : 30 requetes/seconde
500
Erreur interne
Le serveur a rencontre une erreur inattendue.
Actions recommandees
Consultez les logs docker compose logs -f threatclaw-core
Redemarrez le service docker compose restart threatclaw-core
Verifiez l'espace disque et la RAM disponible
502
Passerelle indisponible
Nginx n'arrive pas a joindre ThreatClaw. Le core est peut-etre en cours de redemarrage.
Actions recommandees
Verifiez que les services tournent docker compose ps
Redemarrez la stack docker compose restart
Les modeles IA mettent ~2 min a charger au demarrage
503
Service indisponible
ThreatClaw est en cours de demarrage ou de mise a jour. Ca devrait se resoudre automatiquement.
Actions recommandees
Attendez 30 secondes et rechargez la page
Premier demarrage : les modeles se telechargent (~18 GB)
Verifiez le health curl -k https://localhost/api/health
504
Timeout passerelle
L'analyse IA a depasse la limite de temps (300s). Le dossier d'incident est peut-etre volumineux.
Actions recommandees
L'investigation est peut-etre encore en cours en arriere-plan
Verifiez les logs docker compose logs threatclaw-core | grep INVESTIGATION
Verifiez qu'Ollama repond docker compose logs ollama
