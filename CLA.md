## How to Sign

Signing is automated via the CLA Assistant bot on GitHub:

1. Submit a Pull Request to the ThreatClaw repository
2. The CLA Assistant bot will comment on your PR
3. Click the link and sign with your GitHub account
4. The bot will update your PR status

You only need to sign once. The signature applies to all future contributions.
