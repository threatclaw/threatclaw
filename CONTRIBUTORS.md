## How to Get Listed

1. Read [`CONTRIBUTING.md`](CONTRIBUTING.md) — contribution guide and
   license model overview
2. Read and abide by the [`Code of Conduct`](CODE_OF_CONDUCT.md) —
   Contributor Covenant 3.0
3. Sign the [`CLA`](CLA.md) — automatic, 1 click via the CLA Assistant
   bot on your first pull request
4. Submit your PR — it gets reviewed, discussed, and merged

Every contributor with a merged pull request is automatically added here.

## Recognition Levels

| Level | Criteria | Benefits |
|---|---|---|
| **Contributor** | 1 merged PR | Listed in this file · Public listing on [threatclaw.io/contributors](https://threatclaw.io/contributors) |
| **Core Contributor** | 3+ merged PRs | All the above · *"Core Contributor"* badge on GitHub · Free ThreatClaw Pro license · Direct input on roadmap priorities |
| **Core Team** | Invited by maintainers | All the above · Commit rights · Co-ownership of project direction |

Security researchers who report a valid vulnerability per
[`SECURITY.md`](SECURITY.md) are credited in the associated advisory
and, with their consent, in this file.
